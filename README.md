## Installation

Install all dependencies.

```
$ npm install
```

## Run the server

Server listening on: http://localhost:8080

```
$ npm start
```

## Build

Build server

```
$ npm run build
```


## Serve

Quickly restart the server without waiting for babel to recompile all files

```
$ npm run serve
```

## Test

Test server

```
$ npm test
```
